# Ticaga
### Timeless Card Games

Ticaga is a modern multiplayer card game platform focused on timeless, strategy-driven card games played in real time with friends or online opponents.

The platform is designed to support live game lobbies, real-time gameplay, chat, and competitive play while providing a clean and responsive web experience.

## Planned Features

- Real-time multiplayer gameplay
- Game lobbies and matchmaking
- Global and room-based chat
- Support for multiple classic card games
- Player profiles and statistics
- Tournament and competitive play support

## Technology Stack

Backend
- .NET Web API
- PostgreSQL
- SignalR for real-time communication

Frontend
- React + TypeScript

Infrastructure
- Docker containerization
- Cloud deployment
